What is NRemedy

NRemedy is a suite of class library for .NET. It builds upon BMC remedy C API and helps to access remedy using programing language in .NET platform like C#. The benifit of NRemedy is that it depends on less native dynamic link library. NRemedy also benifit from many .NET programing features like LINQ,generic-type,reflection,object-oriented. If you are familiar with .NET, and you have to intergrate with remedy in your product or system, it is a choice.

 

Main feature

  • Based on .NET framework 4.0(2.0 is also consider but need rebuild)
  • Access AR form in object-oriented. Support regular,join form, support create,delete,set,query,statistics
  • You can choose LINQ query, NRemedy imeplement IQueryable in LINQ
  • Expose transaction interface in AR
  • Expose Server Information Get
  • Expose Form and Field query API, a code generation tool is supplied.(in developing)

Quick Start

Before any other operation, login is necessary.

ARLoginContext context = null;

try

{

    //login will be perform in the construct of ARLoginContext

    context = new ARLoginContext(TestServer, TestAdmin, TestAdminPwd);

}

catch (Exception ex)

{

               

}

finally

{

    //Dispose will automaticlly logout from AR

    if(context != null)

        context.Dispose();

}

 

If login failed, you can catch a exception.

ARLoginContext context = null;

try

{

    context = new ARLoginContext(TestServer + "something", TestAdmin, TestAdminPwd);

}

catch (ARException ex)

{

    //can not establish the connect of AR

    Assert.AreEqual(90, ex.MessageNumber); //183977-ErrorMsgs-7604.pdf page  72

}

finally

{

    if(context != null)

        context.Dispose();

}

Now, you can finish a simple create and query.

ARLoginContext context = new ARLoginContext(TestServer, TestAdmin, TestAdminPwd);

try

{

    //create a generic type proxy to perform API

    ARProxy<NRemedy_Test_Regular_Form> proxy = new ARProxy<NRemedy_Test_Regular_Form>(context);

    //simply use a model instance to create an entry

    string entryId = proxy.CreateEntry(new NRemedy_Test_Regular_Form {

        CharacterField = TestCharacterFieldValue

    });

    //also use a entryid(request id) to get entry from AR

    NRemedy_Test_Regular_Form model = proxy.GetEntry(entryId);

}

catch (Exception ex)

{

    Assert.AreEqual(null, ex);

}

finally

{

    context.Dispose();

}
